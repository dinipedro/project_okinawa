# Functional User Guide

Complete usage guide for customers and restaurant staff using Project Okinawa.

---

# Part 1: Customer Application Guide

## Getting Started

### Download and Installation

1. Download the app from App Store (iOS) or Google Play (Android)
2. Open the app
3. Create an account or sign in

### Account Creation

1. Tap "Create Account"
2. Enter your information:
   - Full name
   - Email address
   - Phone number
   - Password (minimum 8 characters)
3. Verify your email
4. Complete your profile

### Sign In Options

- **Email/Password**: Enter credentials
- **Google**: Tap "Continue with Google"
- **Apple**: Tap "Continue with Apple" (iOS only)

---

## Finding Restaurants

### Explore Screen

Browse restaurants by:
- **Location**: Uses your GPS to show nearby restaurants
- **Category**: Filter by cuisine type
- **Rating**: Sort by customer ratings
- **Distance**: Sort by proximity

### Search

1. Tap the search icon
2. Enter restaurant name, cuisine type, or dish
3. View results
4. Tap a restaurant to view details

### Restaurant Profile

View information including:
- Photos and gallery
- Menu with prices
- Operating hours
- Location and map
- Customer reviews
- Average wait time

---

## Placing Orders

### At a Restaurant (Dine-In)

1. **Scan QR Code**
   - Scan the QR code on your table
   - The app opens the restaurant's menu

2. **Browse Menu**
   - Categories are organized by type
   - Tap an item to see details
   - View ingredients, allergens, and photos

3. **Add to Cart**
   - Select quantity
   - Add special instructions (no onions, extra sauce, etc.)
   - Tap "Add to Cart"

4. **Review Cart**
   - View all items
   - Modify quantities
   - Remove items if needed
   - See subtotal and taxes

5. **Place Order**
   - Confirm table number
   - Add tip (optional)
   - Select payment method
   - Tap "Place Order"

6. **Track Order**
   - View real-time status:
     - Order received
     - Being prepared
     - Ready for service
     - Served

### Takeout/Delivery

1. Select restaurant
2. Choose "Takeout" or "Delivery"
3. Browse menu and add items
4. Enter delivery address (if delivery)
5. Select pickup time (if takeout)
6. Complete payment
7. Track order status

---

## Making Reservations

### Book a Table

1. Go to restaurant profile
2. Tap "Reserve a Table"
3. Select:
   - Date
   - Time
   - Number of guests
4. Add special requests (high chair, wheelchair accessible, etc.)
5. Confirm reservation

### Manage Reservations

- View upcoming reservations
- Modify date/time (subject to availability)
- Add or reduce guests
- Cancel reservation
- Receive reminders

---

## Digital Wallet

### Add Funds

1. Go to Wallet section
2. Tap "Add Funds"
3. Select amount
4. Choose payment method:
   - Credit/Debit card
   - PIX
   - Bank transfer
5. Confirm transaction

### Payment Methods

- **Wallet Balance**: Fastest, no fees
- **Credit Card**: Visa, Mastercard, American Express
- **Debit Card**: Direct from bank account
- **PIX**: Instant Brazilian payment

### Transaction History

- View all transactions
- Filter by date or type
- Download receipts
- Track spending

---

## Reviews and Ratings

### Write a Review

1. After dining, go to "My Orders"
2. Select the order to review
3. Rate your experience (1-5 stars):
   - Food quality
   - Service
   - Ambiance
   - Value for money
4. Write your comments
5. Add photos (optional)
6. Submit review

### View Reviews

- Read other customers' reviews
- See photos from other diners
- View restaurant response to reviews

---

## Loyalty Program

### Earning Points

- Earn points on every purchase
- Bonus points for:
   - First order at a new restaurant
   - Ordering during off-peak hours
   - Inviting friends

### Redeeming Points

1. Go to Loyalty section
2. View available rewards
3. Select reward
4. Apply at checkout

### Tiers

| Tier | Points Required | Benefits |
|------|-----------------|----------|
| Bronze | 0 | 1 point per R$1 |
| Silver | 500 | 1.5 points per R$1, priority reservations |
| Gold | 2000 | 2 points per R$1, exclusive offers |
| Platinum | 5000 | 3 points per R$1, VIP access |

---

## Settings and Preferences

### Profile Settings

- Update personal information
- Change password
- Manage email preferences
- Update phone number

### Notification Settings

- Order updates
- Reservation reminders
- Promotional offers
- New restaurant alerts

### Dietary Preferences

Set preferences to filter menus:
- Vegetarian
- Vegan
- Gluten-free
- Lactose-free
- Nut allergy
- Other allergies

---

# Part 2: Restaurant Application Guide

## Getting Started

### Account Setup

1. Receive credentials from system administrator
2. Download restaurant app
3. Sign in with provided credentials
4. Select your role

### User Roles

| Role | Access |
|------|--------|
| Waiter | Orders, tables, basic functions |
| Bartender | Bar orders, drink preparation |
| Kitchen | Food orders, KDS access |
| Maître | Reservations, floor plan, all orders |
| Manager | Full access, reports, settings |

---

## Dashboard

### Overview

The dashboard shows:
- Active orders count
- Today's revenue
- Pending reservations
- Table availability
- Staff on duty

### Quick Actions

- View new orders
- Check pending reservations
- See kitchen queue
- View floor plan

---

## Order Management

### Receiving Orders

When new order arrives:
1. Notification sound plays
2. Order appears in queue
3. View order details:
   - Table number
   - Items ordered
   - Special instructions
   - Order time

### Updating Order Status

1. **Received**: Order acknowledged
2. **Preparing**: Kitchen/bar working on it
3. **Ready**: Ready for service
4. **Served**: Delivered to customer
5. **Completed**: Payment received

### Order Actions

- Mark items as ready individually
- Add notes for kitchen
- Split order between tables
- Transfer order to different table
- Cancel items (with authorization)

---

## Kitchen Display System (KDS)

### Interface

- Orders displayed by arrival time
- Color coding by urgency:
   - White: Normal
   - Yellow: 10+ minutes
   - Red: 15+ minutes
- Category filters (food, drinks, desserts)

### Workflow

1. View incoming orders
2. Tap to acknowledge
3. Prepare items
4. Mark individual items ready
5. Complete order

### Bump Bar Actions

- Swipe right: Mark ready
- Swipe left: View details
- Tap: Select order

# Guia Funcional do Usuário

Guia completo de uso para clientes e equipe de restaurantes usando o Project Okinawa.

---

# Parte 1: Guia do Aplicativo do Cliente

## Começando

### Download e Instalação

1. Baixe o app na App Store (iOS) ou Google Play (Android)
2. Abra o aplicativo
3. Crie uma conta ou faça login

### Criação de Conta

1. Toque em "Criar Conta"
2. Insira suas informações:
   - Nome completo
   - E-mail
   - Telefone
   - Senha (mínimo 8 caracteres)
3. Verifique seu e-mail
4. Complete seu perfil

---

## Encontrando Restaurantes

### Tela de Exploração

Navegue por restaurantes por:
- **Localização**: Usa seu GPS para mostrar restaurantes próximos
- **Categoria**: Filtre por tipo de culinária
- **Avaliação**: Ordene por avaliações
- **Distância**: Ordene por proximidade

### Busca

1. Toque no ícone de busca
2. Digite nome do restaurante, tipo de culinária ou prato
3. Veja os resultados
4. Toque em um restaurante para ver detalhes

---

## Fazendo Pedidos

### No Restaurante (Presencial)

1. **Escaneie o QR Code**
   - Escaneie o QR code na sua mesa
   - O app abre o cardápio do restaurante

2. **Navegue pelo Cardápio**
   - Categorias organizadas por tipo
   - Toque em um item para ver detalhes

3. **Adicione ao Carrinho**
   - Selecione quantidade
   - Adicione instruções especiais
   - Toque em "Adicionar ao Carrinho"

4. **Revise o Carrinho**
   - Veja todos os itens
   - Modifique quantidades
   - Veja subtotal e taxas

5. **Faça o Pedido**
   - Confirme número da mesa
   - Adicione gorjeta (opcional)
   - Selecione método de pagamento
   - Toque em "Fazer Pedido"

6. **Acompanhe o Pedido**
   - Status em tempo real

---

## Fazendo Reservas

### Reserve uma Mesa

1. Vá ao perfil do restaurante
2. Toque em "Reservar Mesa"
3. Selecione:
   - Data
   - Horário
   - Número de pessoas
4. Adicione pedidos especiais
5. Confirme reserva

### Gerenciar Reservas

- Ver reservas futuras
- Modificar data/horário
- Cancelar reserva
- Receber lembretes

---

## Carteira Digital

### Adicionar Fundos

1. Vá à seção Carteira
2. Toque em "Adicionar Fundos"
3. Selecione valor
4. Escolha método de pagamento
5. Confirme transação

### Métodos de Pagamento

- **Saldo da Carteira**: Mais rápido
- **Cartão de Crédito**: Visa, Mastercard
- **Cartão de Débito**: Direto da conta
- **PIX**: Pagamento instantâneo

---

## Avaliações

### Escrever Avaliação

1. Após refeição, vá em "Meus Pedidos"
2. Selecione o pedido para avaliar
3. Avalie (1-5 estrelas)
4. Escreva seus comentários
5. Adicione fotos (opcional)
6. Envie avaliação

---

## Programa de Fidelidade

### Ganhando Pontos

- Ganhe pontos em cada compra
- Pontos bônus por primeiro pedido
- Pontos bônus por indicar amigos

### Resgatando Pontos

1. Vá à seção Fidelidade
2. Veja recompensas disponíveis
3. Selecione recompensa
4. Aplique no checkout

---

# Parte 2: Guia do Aplicativo do Restaurante

## Começando

### Configuração da Conta

1. Receba credenciais do administrador
2. Baixe o app do restaurante
3. Faça login com credenciais fornecidas
4. Selecione seu papel

### Papéis de Usuário

| Papel | Acesso |
|-------|--------|
| Garçom | Pedidos, mesas, funções básicas |
| Barman | Pedidos de bebidas, preparação |
| Cozinha | Pedidos de comida, acesso ao KDS |
| Maître | Reservas, planta, todos os pedidos |
| Gerente | Acesso total, relatórios, configurações |

---

## Dashboard

### Visão Geral

O dashboard mostra:
- Contagem de pedidos ativos
- Receita do dia
- Reservas pendentes
- Disponibilidade de mesas
- Equipe de plantão

---

## Gestão de Pedidos

### Recebendo Pedidos

Quando novo pedido chega:
1. Som de notificação toca
2. Pedido aparece na fila
3. Veja detalhes do pedido

### Atualizando Status

1. **Recebido**: Pedido reconhecido
2. **Preparando**: Cozinha/bar trabalhando
3. **Pronto**: Pronto para servir
4. **Servido**: Entregue ao cliente
5. **Concluído**: Pagamento recebido

---

## Sistema de Display de Cozinha (KDS)

### Interface

- Pedidos exibidos por tempo de chegada
- Código de cores por urgência:
   - Branco: Normal
   - Amarelo: 10+ minutos
   - Vermelho: 15+ minutos

### Fluxo de Trabalho

1. Ver pedidos recebidos
2. Tocar para reconhecer
3. Preparar itens
4. Marcar itens individuais como prontos
5. Completar pedido

---

## Gestão de Planta do Salão

### Status das Mesas

| Status | Cor | Significado |
|--------|-----|-------------|
| Disponível | Verde | Pronta para sentar |
| Ocupada | Vermelho | Clientes sentados |
| Reservada | Azul | Reserva futura |
| Limpeza | Amarelo | Sendo preparada |

### Ações nas Mesas

- Sentar clientes
- Transferir mesa
- Juntar mesas
- Ver histórico
- Gerar conta

---

## Gestão de Reservas

### Reservas do Dia

- Ver lista cronológica
- Ver tamanho do grupo e pedidos especiais
- Check-in de clientes
- Marcar no-shows

### Ações de Reserva

1. **Confirmar**: Verificar reserva
2. **Modificar**: Mudar data/hora/tamanho
3. **Cancelar**: Com motivo
4. **Check-in**: Marcar chegada
5. **Sentar**: Atribuir mesa

---

## Relatórios Financeiros

### Resumo Diário

- Receita total
- Contagem de pedidos
- Ticket médio
- Breakdown de pagamentos

### Relatórios Detalhados

- Vendas por categoria
- Itens mais vendidos
- Análise de horários de pico
- Performance da equipe

---

## Distribuição de Gorjetas

### Gorjetas Diárias

- Ver total de gorjetas coletadas
- Breakdown por turno
- Gorjetas individuais por garçom

### Métodos de Distribuição

- Divisão igual
- Baseado em pontos (horas trabalhadas)
- Baseado em performance

---

## Fluxos Comuns

### Abertura de Turno

1. Bater ponto
2. Verificar reservas do dia
3. Revisar planta do salão
4. Verificar disponibilidade do cardápio
5. Checar alertas de estoque

### Durante o Serviço

1. Monitorar pedidos recebidos
2. Atualizar status dos pedidos
3. Gerenciar fluxo de mesas
4. Lidar com reservas
5. Processar pagamentos

### Fechamento de Turno

1. Completar todos os pedidos
2. Fechar comandas
3. Gerar relatório diário
4. Revisar gorjetas
5. Bater ponto de saída
